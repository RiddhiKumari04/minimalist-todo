# Minimalist To-Do App (MERN + Tailwind)

## Features
- Clean, distraction-free UI
- Task prioritization
- Optional reminders
- Offline support (basic service worker caching)

## Tech Stack
- React + Vite + Tailwind
- Node.js + Express
- MongoDB + Mongoose

## Setup

### Backend
```bash
cd server
cp .env.example .env   # fill in your Mongo URI
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```

Frontend: http://localhost:3000  
Backend: http://localhost:4000
